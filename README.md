Project 2: Public Web Access using ngrok

 Overview
This project demonstrates how to expose a local web server running on Raspberry Pi to the internet using ngrok.

 Objective
To access a locally hosted web application from anywhere using a public URL.

 Tools Used
- Raspberry Pi
- ngrok
- Web server (port 80)

Procedure
1. Login to ngrok account.
2. Install ngrok on Raspberry Pi.
3. Add ngrok authentication token.
4. Start local web server.
5. Run ngrok to expose port 80.
6. Access the generated public URL.

 Web Link
https://semiyearly-anglea-unburdensome.ngrok-free.dev/

 Result
The local web server running on Raspberry Pi was successfully exposed to the internet using ngrok.
